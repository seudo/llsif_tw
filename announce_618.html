Android OS 6.0 相關注意

Date 2015-10-06 二 Tags POST

關於近日將預定發布的Android OS 6.0，
現在「Love Live!學園偶像祭」還無法正確地對應，
若更新至Android OS 6.0 後，可能無法正常進行遊戲。

對於Android OS 6.0的對應處理，未來將預定發布更新修正檔。
在此先與各位玩家道歉，
在完全對應Android OS 6.0的更新修正完了之前，
尚請各位玩家先不要進行相關的OS更新。
未來若有相關修正更新，將再另外通知玩家。

造成玩家不便的地方，再次請玩家包涵。
以後也請多多支持。
